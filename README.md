# myshoppinglist-backend
MyShoppingList ist eine Web-Anwendung, mit der Nutzer Produkte an einem zentralen Ort speichern können, die sie in Zukunft kaufen möchten oder müssen. Ziel ist es, einen besseren Überblick über geplante Käufe zu behalten, ohne diese auf verschiedenen Anwendungen verteilen zu müssen. Zudem ermöglicht die Anwendung die Organisation der Produkte in unterschiedlichen Kategorien.

## Features
- Erstellen und Anzeigen von Items
- Kategorisierung von Produkten (z.B. Fashion, Beauty, Lebensmittel, Bücher etc.)
- Speicherung von Produktinfos (Preis, URL und Kaufstatus)
- Übersicht über geplante und bereits gekaufte Produkte

## Aktueller Stand
Aktuell implementiert die Anwendung eine GET-Route (`/items`), die eine Liste von Beispiel-Items zurückgibt.
Weitere Funktionen wie z.B. das Erstellen, Löschen und Speichern von Items werden im Verlauf des Projekts ergänzt.
